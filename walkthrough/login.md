# Login to RepoLens

Welcome to RepoLens! To unlock the full power of the extension, you need to log in.

## Steps

1. Open the RepoLens activity bar.
2. Click the **Login** button or run the command `RepoLens: Login` from the Command Palette.
3. A browser window will open and ask you to authenticate.
4. Complete the login flow using your RepoLens account.
5. Once authenticated, return to VS Code and confirm that the RepoLens view shows you as logged in.

## Why login?

- Enables AI-based code reviews
- Allows rule scanning and refactoring
- Activates GitHub PR integration
- Lets RepoLens remember your preferences

If you don’t have an account yet, use the browser page to sign up and then return to VS Code to complete login.
