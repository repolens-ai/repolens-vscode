# Explore RepoLens

Open the RepoLens tutorial and see how to use the extension features.

## What you can do

- Open a guided tutorial file
- Learn how code reviews work in the IDE
- See where the chat assistant appears
- Discover how to toggle code lens support

## Try it now

1. Run the command `RepoLens: Open welcome file`.
2. Read the walkthrough content in the opened file.
3. Use the provided commands and links to continue exploring.

RepoLens is designed to help you review code, ask questions, and make changes without leaving VS Code.
