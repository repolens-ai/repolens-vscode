# Review Your PRs with RepoLens

Connect RepoLens to GitHub to get automated reviews on pull requests.

## Why GitHub code reviews?

- Automatically review new pull request changes
- Get AI-powered suggestions on diffs
- Reduce review latency for your team

## How to start

1. Install the RepoLens GitHub App from the installation page.
2. Authorize your repositories.
3. Open a pull request in VS Code or GitHub.
4. RepoLens will provide review guidance directly in the PR workflow.

This integration helps you catch issues earlier and keeps reviews consistent.
