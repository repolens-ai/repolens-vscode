# Configure RepoLens

Create a `.repolens.yaml` file to customize RepoLens behavior for your project.

## Common settings

- `python_version`: Specify the Python version used by the project.
- `ruleType.refactorings`: Enable or disable refactoring suggestions.
- `ruleType.comments`: Enable or disable inline comments.

## How to create a config file

1. Run the command `RepoLens: Create a default RepoLens configuration file`.
2. Open `.repolens.yaml` in your workspace.
3. Edit the settings to match your project needs.

Configuring RepoLens helps the extension make better suggestions and behave consistently across your repo.
