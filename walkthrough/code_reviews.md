# Get Feedback on Your Current Changes

RepoLens can review your open files, current branch, or uncommitted changes.

## Steps

1. Open the RepoLens view in the activity bar.
2. Use the command palette or the RepoLens sidebar to start a code review.
3. Review suggestions, refactorings, and AI feedback.

## Why this helps

- Get fast feedback on your current work
- Find issues before you commit
- Improve code quality with AI-guided suggestions

Tip: Run a review after editing a file to catch improvements early.
