# Line Suggestions with RepoLens

RepoLens can suggest improvements one line at a time while you edit.

## How it works

- RepoLens analyzes the current file and highlights suggestions on individual lines.
- It also passively reviews your open code and recommends improvements for quality, readability, and maintainability.
- Suggestions can include refactorings, style improvements, or code fixes.
- You can accept suggestions directly from the RepoLens sidebar or code lens actions.

## Try it now

1. Open a file in your workspace.
2. Make an edit or open an existing change.
3. Look for line suggestions in the RepoLens view or inline code lens hints.
4. Apply the suggestion if it improves your code.

Line suggestions help you move faster and keep your code clean without leaving the editor.
